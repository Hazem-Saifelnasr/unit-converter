# GitHub Pages Privacy Policy Checklist

## 1. Prepare The Files

- [x] Confirm `docs/PRIVACY_POLICY.md` contains the final policy text.
- [x] Confirm `release-assets/privacy-policy/index.html` matches the final policy text.
- [x] Commit the privacy policy files to the repository.
- [x] Push the branch to GitHub.

## 2. Enable GitHub Pages

- [x] Open the repository on GitHub.
- [x] Open `Settings` -> `Pages`.
- [x] Set `Source` to `Deploy from a branch`.
- [x] Select the publish branch, usually `main`.
- [x] Select the `/root` folder.
- [x] Save the GitHub Pages settings.

## 3. Verify The Public URL

- [x] Wait for the Pages deployment to finish.
- [x] Open the GitHub Pages site URL.
- [x] Open `/release-assets/privacy-policy/` on that site.
- [x] Confirm the page loads without login.
- [x] Confirm the page shows the current support email.

## 4. Use It In Play Console

- [x] Copy the final public privacy policy URL.
- [x] Paste it into Google Play Console `Privacy policy`.
- [x] Save the Play Console section.
