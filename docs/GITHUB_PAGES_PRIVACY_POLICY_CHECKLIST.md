# GitHub Pages Privacy Policy Checklist

## 1. Prepare The Files

- [ ] Confirm `docs/PRIVACY_POLICY.md` contains the final policy text.
- [ ] Confirm `release-assets/privacy-policy/index.html` matches the final policy text.
- [ ] Commit the privacy policy files to the repository.
- [ ] Push the branch to GitHub.

## 2. Enable GitHub Pages

- [ ] Open the repository on GitHub.
- [ ] Open `Settings` -> `Pages`.
- [ ] Set `Source` to `Deploy from a branch`.
- [ ] Select the publish branch, usually `main`.
- [ ] Select the `/root` folder.
- [ ] Save the GitHub Pages settings.

## 3. Verify The Public URL

- [ ] Wait for the Pages deployment to finish.
- [ ] Open the GitHub Pages site URL.
- [ ] Open `/release-assets/privacy-policy/` on that site.
- [ ] Confirm the page loads without login.
- [ ] Confirm the page shows the current support email.

## 4. Use It In Play Console

- [ ] Copy the final public privacy policy URL.
- [ ] Paste it into Google Play Console `Privacy policy`.
- [ ] Save the Play Console section.
